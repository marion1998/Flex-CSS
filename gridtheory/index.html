The Grid Theory

Grid est un ensemble de propriétés CSS qui permet la disposition d'éléments dans une grille sur une page web. Les premières spécifications de CSS Grid ont été proposées par Microsoft en 2011. Comme son nom l'indique, on parle d'une grille en deux dimensions. Les lignes et les colonnes ainsi que leur comportement sont définies entièrement en CSS.

display

La propriété "display" permet de définir un container comme un grid container, c'est à dire qu'il sera possible d'utiliser grid à l'intérieur de ce container pour disposer des différents éléments comme bon nous semble.

Pour voir le placement par défaut de "grid", il suffit de créer une grille avec un ensemble d'objets. Sans fournir aucune information de placement, ces objets se placeront chacun sur une cellule de la grille.

.container

display: | ;

grid-template-columns: 1fr 1fr 1fr 1fr;

grid-template-rows: 1fr 1fr 1fr;

}

grid-template-columns

"grid-template-columns" permet de crée des colonnes.

On peut choisir la largeur de chacune de nos colonnes (definis par des px (pixels), % (poucentage), fr (fraction) ou auto (automatic) ). Ainsi 1fr 1fr 1fr ou repeat(3, 1fr) permettent d'obtenir une grille de 3 colonnes faisant chacune 1/3 de l'espace disponible restant (après calcul des margins et paddings).

.container {

display: grid;

grid-template-columns: | | | | ;

grid-template-rows: 1fr 1fr 1fr;

}

grid-template-areas

La propriété "grid-template-areas" permet de définir des zones de grille nommées comme on le souhaite. Chaque cellule a un nom qui peut être attribué en css.

.container {

display: grid;

grid-template-areas: |

;}

div.A { grid-area : un; }

div.B { grid-area : deux; }

div.C { grid-area : trois; }

div.D { grid-area : quatre; }

gaps

"gaps" va nous servir à définir des espaces entre les cellules que comporte la grille, ca peut s'avérer utile si l'on veux aérer notre page.

.container {

display: grid;

grid-template-columns: 1fr 1fr 1fr 1fr;

grid-template-rows: 1fr 1fr 1fr;

row-gap: value ;

column-gap: value ;

gap: value ;

}
