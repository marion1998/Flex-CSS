The Grid Theory

Le module CSS Grid Layout est un module assez jeune intégré à CSS3 blablabla

display

.container {

display: grid | none ;

}



grid-template-columns

.container {

grid-template-columns: none | line name | track size | auto ;

}



grid-template-rows

.container {

grid-template-rows: none | line name | track size | auto ;

}



grid-template-areas

.container {

grid-template-areas: none | string ;

}




gaps

.container {

row-gap: value ;

column-gap: value ;

gap: value ;

}
