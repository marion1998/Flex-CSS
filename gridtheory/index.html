The Grid Theory

Grid est un ensemble de propriétés CSS qui permet la disposition d'éléments dans une grille sur une page web. Les premières spécifications de CSS Grid ont été proposées par Microsoft en 2011.

display

La propriété "display" permet de définir un container comme un grid container, c'est à dire qu'il sera possible d'utiliser grid à l'intérieur de ce container pour disposer des différents éléments comme bon nous semble.

Pour voir le placement par défaut de "grid", il suffit de créer une grille avec un ensemble d'objets. Sans fournir aucune information de placement, ces objets se placeront chacun sur une cellule de la grille

.container

display: grid | block ;

grid-template-columns: 1fr 1fr 1fr 1fr;

grid-template-rows: 1fr 1fr 1fr;

}

grid-template-columns

"grid-template-columns" permet de crée des colonnes dans lesquels ont pourras y placer des elements (tel que du texte).

On peut ainsi les placer au début, à la fin ou au milieu de cet axe. On peut choisir l'epaisseur de chacune de nos colonnes (definis par des px (pixels), % (poucentage), fr (fraction) ou auto (automatic) )

.container {

display: grid;

grid-template-columns: | | | | ;

grid-template-rows: 1fr 1fr 1fr;

}

grid-template-rows

La propriété "grid-template-columns" permet de crée des lignes dans lesquels ont pourras également y placer des elements (qui pourrait aussi être du texte).

grid-template-columns: 1fr 1fr 1fr 1fr;

grid-template-rows: | | | | ;

grid-template-areas

La propriété "grid-template-areas" permet de permet de définir des zones de grille nommées a notre de la facons que l'on souhaite

.container {

display: grid;

grid-template-areas: none | string ;

}

gaps

"gaps" va nous servir a definir des espaces entre les cellules que comporte la grille, ca peut s'averer utile si l'on veux aérée notre page.

.container {

display: grid;

grid-template-columns: 1fr 1fr 1fr 1fr;

grid-template-rows: 1fr 1fr 1fr;

row-gap: value ;

column-gap: value ;

gap: value ;

}

Placing Items

Be careful with those, only use to define what we call a gridcell

An automatic placement for items that aren't placed

grid-row

.item {

grid-row-start : line name | line number ;

grid-row-end : line name | line number ;

grid-row-: ;

}

grid-column

.item {

grid-column-start : line name | line number ;

grid-column-end : line name | line number ;

grid-column: ;

}

Auto Placement - grid-auto-flow

An automatic placement for items that aren't placed

.item {

grid-auto-flow : row ;

grid-auto-flow : column ;

grid-auto-flow : dense ;

}
