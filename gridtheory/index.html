The Grid Theory

Grid est un ensemble de propriétés CSS qui permet la disposition d'éléments dans une grille sur une page web. Les premières spécifications de CSS Grid ont été proposées par Microsoft en 2011. Comme son nom l'indique, on parle d'une grille en deux dimensions. Les lignes et les colonnes ainsi que leur comportement sont définis entièrement en CSS. Il faut quand même garder en tête que gride et flex ne sont absolument pas concurrent mais bien complementaire,

display

La propriété "display" permet de définir un container comme un grid container, c'est à dire qu'il sera possible d'utiliser grid à l'intérieur de ce container pour disposer des différents éléments comme bon nous semble.

Pour voir le placement par défaut de "grid", il suffit de créer une grille avec un ensemble d'objets. Sans fournir aucune information de placement, ces objets se placeront chacun sur une cellule de la grille

.container

display: | ;

grid-template-columns: 1fr 1fr 1fr 1fr;

grid-template-rows: 1fr 1fr 1fr;

}

grid-template-columns

"grid-template-columns" permet de crée des colonnes dans lesquels ont pourras y placer des elements (tel que du texte).

On peut ainsi les placer au début, à la fin ou au milieu de cet axe. On peut choisir l'epaisseur de chacune de nos colonnes (definis par des px (pixels), % (poucentage), fr (fraction) ou auto (automatic) ). Ainsi 1fr 1fr 1fr ou repeat(3, 1fr) permettent d'obtenir une grille de 3 colonnes faisant chacune 1/3 de l'espace disponible restant (après calcul des margins et paddings).

.container {

display: grid;

grid-template-columns: | | | | ;

grid-template-rows: 1fr 1fr 1fr;

}

grid-template-areas

La propriété "grid-template-areas" permet de définir des zones de grille nommées de la façon que l'on souhaite. Chaque cellule a un nom , qui peut être attribué , en css.

.container {

display: grid;

grid-template-areas: |

;}

div.A { grid-area : un; }

div.B { grid-area : deux; }

div.C { grid-area : trois; }

div.D { grid-area : quatre; }

Placing Items

Be careful with those, only use to define what we call a gridcell

An automatic placement for items that aren't placed

grid-row-start / grid-row-end

.item {

grid-row-start : | | | ;

grid-row-end : | | | ;

grid-row : | | |

}

grid-column-start / grid-column-end

.item {

grid-column-start : line name | line number ;

grid-column-end : line name | line number ;

}

grid-area

.item {

grid-area : row ;
